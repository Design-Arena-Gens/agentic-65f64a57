## منصة أساسيات التمريض

تطبيق ويب تفاعلي باللغة العربية يساعد المبتدئين على تعلم أهم مهارات ومفاهيم التمريض من خلال وحدات تعليمية مختصرة، قوائم مراجعة عملية، اختبارات سريعة، وخطة مذاكرة يومية واضحة.

### المزايا الرئيسية

- واجهة عربية بالكامل مع دعم اتجاه الكتابة من اليمين لليسار.
- خطة تعلم من ستة أسابيع تغطي الأمان، التقييم، الرعاية اليومية، الأدوية، والتواصل.
- قائمة مهارات تفاعلية مع نصائح فورية، واختبار متعدد الخيارات مع تفسير للإجابات.
- مصادر قابلة للتحميل وجدولة مذاكرة يومية لمدة ساعة.

### التشغيل محلياً

```bash
npm install
npm run dev
```

بعد تشغيل الخادم افتح [http://localhost:3000](http://localhost:3000) للاطلاع على التطبيق.

### بناء للإنتاج

```bash
npm run lint
npm run build
```

سيقوم `next build` بتجهيز نسخة إنتاجية جاهزة للنشر على Vercel أو أي مزود استضافة متوافق.
